Complete referral/affiliate system with seasonal campaigns and WooCommerce integration.

== Description ==
Provides affiliate accounts, seasonal campaigns, referral tracking, and payout management with WooCommerce support.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/kibgim-referrals` directory or install via WordPress.
2. Activate the plugin through the 'Plugins' screen.
3. Use shortcode `[kgr_affiliate_optin]` for opt-in form and `[kgr_affiliate_dashboard]` for dashboard.
